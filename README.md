## 📚 Tech Stacks

- React + TypeScript + Vite
- Tailwind CSS for styling
- SWR for data fetcher
- Vitest for unit testing

## ✨ Features

- See all available foods
- Filter by category
- Search by keyword
- Shareable filter URL
